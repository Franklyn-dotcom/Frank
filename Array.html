Example

Array elements are numbered, starting with zero.

We can get an element by its number in square brackets:

var streetName = ["Argonne St", "Bunyard", "Lisbon"]; alert ( streetName[0] ); //New York alert ( streetName[1] ); // Boston alert ( streetName[2] ); // Lisbon

We can replace an element:

streetName[2] = "Berlin"; //now["Argonne St", "Bunyard", "Berlin"]

Or add a new one to the array

streetName[3] = "Lagos"; //now["Argonne St", "Bunyard", "Berlin", "Lagos"]

The total count of the elements in the array is its length:

var fruits = ["Apple", "Orange", "Plum"]; alert ( fruits.length ); // 3

fruits = ["Banana", "Orange", "Apple", "Mango"]; var last = fruits [fruits.length - 1];

Array Methods

Here, we are going to show you few examples on array methods

pop()

So we are going to start with arr.pop()

pop() ‐ It removes and returns the last element of an array.

Here are some examples of arr.pop

var clothes = ["pyjamas", "dungarees", "jumsuit"]; alert (clothes.pop(2)); // jumsuit alert (clothes); // pyjamas, dungarees

var clothes = ["pyjamas", "dungarees", "jumsuit"]; alert (clothes.pop(0)); // pyjamas alert (clothes); // jumsuit, dungarees

push()

It adds one or more elements to the end of an array.

var clothes = ["pyjamas", "dungarees", "jumsuit"]; alert (clothes.push("polo")); // length is 4 alert (clothes); // jumsuit, dungarees, pyjamas, polo

shift()

This method removes the first element in an array.

var cars = ["hyundai", "ford", "toyota"]; alert (clothes.shift()); // hyundai alert (clothes); // ford, toyota

unshift()

It adds one or more elements in the beginning of the given array.

var fruits = [ "oranges", "Apple" ]; fruits.unshift ( "Pear" ) alert (fruits);
